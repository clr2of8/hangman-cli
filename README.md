
# hangman-cli

This repository contains two command-line Hangman games:

- **hangman.py**: A Python CLI game with pirate-themed ASCII art and dialogue.
- **hangman.ps1**: A PowerShell CLI game (no ASCII art) with standard prompts.

Both games use a shared word list in `words.txt`. To add or change the words for both games, simply edit `words.txt`.

## How to Play

### Python Version
Run with:
```
python3 hangman.py
```

### PowerShell Version
Run with:
```
pwsh hangman.ps1
```

## Features
- Pirate-themed experience in Python
- Shared word list for both games
- Replay option after each round

Enjoy playing Hangman in your favorite shell!
